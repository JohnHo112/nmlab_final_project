# IOTA-DID

## Holder
```
cd iota-holder
```

### Install Modules
```
npm run install-holder-frontend
npm run install-holder-backend
```
### Run frontend
```
npm run holder-frontend
```
### Run backend
```
npm run holder-backend
```
